todo-app-bmad — Design Directions

Direction 1: Clean Minimal

The purest expression of our design philosophy. Rounded input with subtle gray background, circle checkboxes, hairline dividers. Maximum white space, minimum visual noise. The content IS the interface.

Direction 2: Soft Cards

Each todo lives in its own soft card with rounded corners. Centered title, inset input field, green completion checkmarks. Delete buttons appear on hover. Feels tactile and modern.

Direction 3: Borderless

Ultra-minimal. Uppercase muted title, underline-only input, no borders between items. Small circle checkboxes in dark on completion. The delete button is invisible until hover. Closest to "just a list on paper."

Direction 4: Bold Header

Dark header creates a strong visual anchor. The input field lives inside the header with a frosted-glass feel. The list below is clean white. Creates clear visual separation between "input mode" and "list mode."

Direction 5: Input + Add Button

Adds a visible "Add" button next to the input field. More explicit affordance for adding tasks — user doesn't need to discover that Enter works. Square checkboxes. Traditional but clear.

Direction 6: Warm & Friendly

Warm amber palette instead of cool blue. Feels inviting and personal — like a notepad on a sunny desk. Softer, more approachable personality. Trades "professional trust" for "personal warmth."
